# Tooling

## Current

- **Bun** — runtime, package manager, test runner
- **Turborepo** — monorepo task orchestration
- **TypeScript** — type checking
- **Drizzle Kit** — database migrations

## Planned (to be added manually)

- **Biome** — linting and formatting (replaces ESLint + Prettier)
- **Conventional commits** — commit message enforcement via commitlint + husky
- **Duplication detection** — code duplication detection tool (jscpd or similar)
- **Shared tsconfig** — base tsconfig for packages to extend
- **No-build packages** — packages configured to use source directly during development (no need to build packages before running apps)
